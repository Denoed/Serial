
# Building

How to build the project / native library.

<br>

## Prerequisites

-   Have **Deno** installed.

-   Have the project cloned.

-   Have GCC compiler installed.

<br>

## Steps

-   Build the native binary.

    ```sh
    deno task compile
    ```

-   Test the code.

    ```sh
    Test/Run.js
    ```
